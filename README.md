# Project 1 – MIMIC-III Tabular Medical Data

This project analyzes tabular medical data and explores in-hospital mortality prediction.

The notebook includes:

- Descriptive statistics
- Distribution analysis
- Classification models
- Regression models
- PCA and t-SNE
- K-Means clustering
- A baseline deep neural network
- An improved mortality-prediction model

The dataset and original course assignment are not included.
